# Code Description: Cafe Menu Website

This is a **simple HTML-based cafe menu website** for "Camper Cafe." Here's what the code does:

## Structure & Content

**HTML Document Setup:**
- Creates a responsive webpage with proper meta tags for character encoding and viewport settings
- Links to an external CSS file (`style11.css`) for styling

**Main Sections:**

1. **Header**
   - Displays the cafe name: "CAMPER CAFE"
   - Shows establishment year: "Est. 2020"
   - Includes decorative horizontal line separator

2. **Coffee Menu Section**
   - Displays a coffee icon image
   - Lists 5 coffee types with prices:
     - French Vanilla ($3.00)
     - Caramel Macchiato ($3.75)
     - Pumpkin Spice ($3.50)
     - Hazelnut ($4.00)
     - Mocha ($4.50)

3. **Desserts Menu Section**
   - Displays a pie/dessert icon image
   - Lists 4 dessert items with prices:
     - Donut ($1.50)
     - Cherry Pie ($2.75)
     - Cheesecake ($3.00)
     - Cinnamon Roll ($2.50)

4. **Footer**
   - Contains a clickable link to a LinkedIn profile (the cafe owner's social media)
   - Displays the cafe's address: "123 Free Code Camp Drive"
   - Includes a bottom decorative line separator

## Key Features

- **Semantic HTML**: Uses proper tags like `<main>`, `<section>`, `<article>`, `<footer>`, and `<address>` for better structure
- **Responsive Design**: Includes viewport meta tag for mobile compatibility
- **External Images**: Fetches coffee and pie icons from FreeCodeCamp CDN
- **Clean Layout**: Uses CSS classes (`item`, `flavor`, `price`, `dessert`) for styling via the external CSS file

The styling (colors, fonts, layout) is handled by the `style11.css` file, which is not shown here.
